# conversor-de-temperatura-python <img width='50' heigth='50' src="https://cdn.jsdelivr.net/gh/devicons/devicon@latest/icons/python/python-original.svg" />
## Olá!
#### Neste repositório, compartilho o resultado do meu aprendizado nas aulas de lógica de programação do curso de Residência em TIC 20. O projeto consiste em um conversor de unidades métricas de temperatura, complementado por uma interface gráfica básica desenvolvida com a biblioteca PySimpleGUI, que visa aprimorar a experiência e a usabilidade do usuário.

#### Além disso, incluí uma funcionalidade adicional que permite visualizar a temperatura em tempo real de um local escolhido pelo usuário. Para isso, utilizei a API da OpenWeather, que fornece dados meteorológicos atualizados com base em coordenadas ou no nome da cidade.

#### Vale ressaltar que tive cerca de 3 dias para finalizar este projeto. Caso encontre algum erro ou tenha sugestões para melhorias, sinta-se à vontade para abrir uma issue; farei as alterações assim que possível.
